# Configura-oPlataformaAzure
Criação de maquina virtual e banco de dados


MAQUINAS VIRTUAIS - Na criação de maquinas virtuais é necessario te atenção, visto que esse processo é repleto de detalhes que podem influenciar no desempenho da maquina. Na configuração é possivel escolher o sistema operacional, disco, rede entre outros. 

BANCO DE DADOS - Quando se trata de banco de dados, deve-se se ter atenção ao servidor que sera utilizado e redundancia do armazenamento. Assim como na criação de maquinas virtuais, enquanto é feito o a configuração, a plataforma informa os valores de quanto aquela configuração custara.
